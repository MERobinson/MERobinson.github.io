Intro to Bioinformatics

1.5 System Setup

1.5.1 Operating system

Most bioinformatics tools are designed for UNIX systems - in other words Mac or Linux. If you need to use a Windows machine I’d recommend running Bio-Linux from a USB stick. This provides you with a Linux workstation pre-setup with many common bioinformatic tools.

1.5.2 R

R is a statistical programming language commonly used for down-stream statistical analysis and visualisation, which is exactly what we’ll be doing in this chapter!

1.5.3 RStudio

R can be run from the command line or using the popular user-interface (or intergrated-development-environment (IDE) if you want to be fancy) RStudio. I’d recommend beginners use RStudio as it’s more intuitive for anyone not familiar with the command line.

To get RStudio simply go here, download the installer appropriate for your system and follow the instructions to install.

1.5.4 SRA-Toolkit

  • For Ubuntu / Bio-Linux:

Run the following command:

wget --output-document sratoolkit.tar.gz http://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/current/sratoolkit.current-ubuntu64.tar.gz
tar -vxzf sratoolkit.tar.gz
  • For Mac OS X:

Run the following command:

curl --output sratoolkit.tar.gz http://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/current/sratoolkit.current-mac64.tar.gz
tar -vxzf sratoolkit.tar.gz
  • For both:

Extract the tar file with the following command:

tar -vxzf sratoolkit.tar.gz
