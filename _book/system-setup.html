2.2 Bio-Linux

Setting up your system with is often one of the most time consuming and frustrating parts of starting out in bioinformatics. To avoid the hassle we’ll be running Bio-Linux from a USB stick. This provides you with a Linux workstation pre-setup with many common bioinformatic tools.

2.3 R

In addition to command line programs we’ll be using R to plot figures. R is a statistical programming language commonly used for down-stream statistical analysis and visualisation. R can be run from the command line, or using the popular user-interface RStudio.

2.3.1 RStudio

We’ll be using RStudio for the course, which isn’t pre-installed on Bio-Linux - so simply click here to download for the Bio-Linux system, open the file and install.
